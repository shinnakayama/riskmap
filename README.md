# Risk mapping

## codes
- `updated_voyage.sql`: query for GFW data to clean up voyages 
- `fishing_prep_trips.sql`: query for GFW data for fishing trips
- `fishing_iuu.py`: model and SHAP value analysis for risk of IUU fishing on fishing vesssels
- `fishing_la.py`: model and SHAP value analysis for risk of labor abuse on fishing vesssels
- `fishing_bin1.sql`: query to sum fishing hours in 1x1 degree grids for mapping 
- `transship_prep_trips.sql`: query for GFW data for trips by carrier vessels
- `transship_prep_trips_loitering.sql`: query for GFW data for trips by carrier vessels with loitering
- `transship_iuu.py`: model and SHAP value analysis for risk of IUU fishing on carrier vesssels
- `transhhip_la.py`: model and SHAP value analysis for risk of labor abuse on carrier vesssels

## data
- `port_risk.csv`: port risk assessment
- `c188.csv`: nations with C188
- `GDP.csv`: GDP by nations
- `HDI.csv`: HDI by nations
- `psma.csv`: nations with PSMA ratification/accession
